# BBAW Akademie Proposal: Companion Dashboard

Live dashboard: https://maximromanov.github.io/bbaw_dashboard/

## Sections

- Headline corpus statistics (PDF volumes, unique works, pages, storage)
- Phase 1 budget calculator (OCR pipeline cost model)
- Corpus-at-a-glance (per-source breakdown, page-length distribution, top works)
- Discipline distributions (Shamela emic, ACO LC, unified 14-bucket comparison)
- Searchable metadata browser (full inventory, ~30,000 records)

## Methodology note

This is a preparatory artifact for proposal review, not a research output.
The underlying inventory aggregates three sources: Arabic Collections Online
(NYU Libraries / NYU Abu Dhabi), the Shamela digital library (ShamelaAY snapshot),
and personal scholarly collections.
